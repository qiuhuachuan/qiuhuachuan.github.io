# 1 创建一个新项目

第一步是创建一个新的 Rasa 项目。执行：

```
rasa init --no-prompt
```

`rasa init`命令会创建 Rasa 项目需要的所有文件，并且在一些示例数据上训练一个简单的 bot。如果忽略`--no-prompt`标记，系统将会逐步询问你有关如何设置项目的一些问题。

|\_\_init\_\_.py|一个空文件，它可以帮助 python 查找你的 actions|
